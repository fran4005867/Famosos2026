# Famosos2026

> Una aplicación web interactiva para explorar y conocer a celebridades destacadas del 2026.

[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)
[![Language](https://img.shields.io/badge/Language-JavaScript-yellow.svg)](https://www.javascript.com/)

## 🌐 Demo en Vivo

Puedes acceder a la aplicación en vivo aquí: **[Famosos2026](https://jsaerospace.github.io/Famosos2026/)**

## 📋 Descripción

Famosos2026 es una plataforma web dinámica que te permite:
- 🎬 Explorar perfiles de celebridades del 2026
- 📸 Ver información detallada y fotografías
- 🌟 Descubrir nuevos talentos y personalidades
- 📱 Acceder desde cualquier dispositivo

## ⚙️ Características

- **Interfaz responsiva**: Compatible con dispositivos móviles, tablets y escritorio
- **Búsqueda dinámica**: Encuentra celebridades rápidamente
- **Información detallada**: Accede a perfiles completos
- **Diseño moderno**: Interfaz intuitiva y atractiva
- **Carga rápida**: Optimizado para el mejor rendimiento

## 🚀 Instalación

### Requisitos previos
- Node.js (versión 14 o superior)
- npm o yarn

### Pasos de instalación

```bash
# Clonar el repositorio
git clone https://github.com/fran4005867/Famosos2026.git

# Navegar al directorio
cd Famosos2026

# Instalar dependencias
npm install

# Ejecutar en modo desarrollo
npm start
```

## 📚 Uso

Una vez que hayas clonado el repositorio e instalado las dependencias:

```bash
# Iniciar servidor de desarrollo
npm start

# Compilar para producción
npm run build

# Ejecutar pruebas
npm test
```

La aplicación se abrirá automáticamente en `http://localhost:3000`
